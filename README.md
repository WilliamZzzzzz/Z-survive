# Z-SURVIVE（末日文字生存）

暗黑写实风的末日丧尸文字生存原型（Web Prototype）。

核心循环：基地（管理 / 制造）→ 出门探索（地图 / 事件 / 战斗）→ 资源回收 → 生存压力（时间 / 状态）

---

## 项目目标（Purpose）

- 类似《死亡日记》《这是我的战争》的末日文字生存玩法
- 强调时间压力、生存状态、探索风险
- 当前阶段目标：完成可玩的系统原型（非商业版本）
